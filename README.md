# Ankita Shendge Portfolio

An interactive frontend portfolio showcasing my experience, technical skills,
education, and selected projects.

The portfolio uses lightweight CSS 3D effects and React interactions to create
an engaging experience without adding a heavy 3D rendering dependency.

## Highlights

- Interactive 3D profile hero with education and resume download
- Responsive technical-skills architecture
- 3D career elevator for exploring professional experience
- Project constellation with connected project nodes and animated details
- Mobile-friendly layouts and reduced-motion support

## Built With

- React
- TypeScript
- Vite
- CSS 3D transforms and animations

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Production Build

```bash
npm run build
npm run preview
```
